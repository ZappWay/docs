ZappWay let you bring data from anywhere (Datsources: Text, PDF, Word, PowerPpoint, Excel, Notion, Airtable, Google Sheets, etc..) into Datastores (container of multiple Datasources)

Then your Datastores can be connected to ChatGPT via Plugins or any other Large Langue Model (LLM) via the ZappWay API.

**ZappWay is made for ChatGPT users, developers, low-coders, no-coders**

![Conversion](/images/zappway-overview.png)

## Vocabulary

| Name       | Description                       |
| ---------- | --------------------------------- |
| Datastore  | Container of multiple Datasources |
| Datasource | Source of data                    |
| LLM        | Large language model              |
| Storage    | Vector database provider          |
